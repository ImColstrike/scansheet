📖 Instructions
1. Ouvrez Google Sheets et créez un nouveau classeur.

2. Allez dans Extensions → Apps Script.

3. Collez ce code dans l'éditeur :
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet() .getSheetByName(e.parameter.sheet || SpreadsheetApp.getActiveSpreadsheet().getSheets()[0].getName()); var row = [e.parameter.timestamp, e.parameter.type, e.parameter.value]; sheet.appendRow(row); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); } 4. Cliquez Déployer → Nouveau déploiement.

5. Choisissez Application Web, accès Tout le monde.

6. Copiez l'URL de déploiement et collez-la ci-dessus.
